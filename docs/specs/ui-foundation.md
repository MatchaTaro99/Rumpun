# UI Foundation

Dokumen ini merangkum wireframe awal, pola navigasi, dan baseline aksesibilitas untuk Rumpun.

## Halaman Utama

- Login
- Dashboard
- Pohon silsilah
- Profil anggota
- Form anggota
- Pencarian
- Pengaturan
- Bantuan

## Aturan Navigasi

- Navigasi utama memakai satu lapisan tab untuk area besar.
- Daftar layar atau alat bantu muncul di lapisan kedua saja.
- Tidak ada navigasi yang lebih dalam dari dua level pada fase desain ini.

## Design Token Awal

- Warna utama diarahkan ke palet hangat yang lembut dan kontrasnya tetap aman.
- Teks dasar dimulai dari 16px.
- Mode teks besar menaikkan ukuran dasar ke 18px untuk membantu pengguna usia lanjut.
- Tombol dan kontrol interaktif minimal 44px tinggi.

## Aksesibilitas

- Gunakan label yang sederhana dan langsung.
- Hindari istilah teknis yang tidak perlu.
- Semua aksi berisiko harus memakai dialog konfirmasi.
- Pesan error, warning, dan success ditulis dengan bahasa yang membantu.

## Pola Komponen

- App shell
- Primary navigation
- Screen selector
- Wireframe preview
- Token swatches
- Confirmation pattern
