# Dokumentasi

Ruang ini menampung dokumen produk, keputusan teknis, dan catatan implementasi awal.

- `architecture.md` menjelaskan susunan layanan dan alur dasar
- `specs/ui-foundation.md` berisi wireframe, token desain, dan pola aksesibilitas
- `adr/` untuk architectural decision record
- `specs/` untuk spesifikasi fitur atau domain
